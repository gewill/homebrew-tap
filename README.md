# Gewill Tap

## How do I install these formulae?

`brew install gewill/tap/<formula>`

Or `brew tap gewill/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gewill/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
